Machine-learning-assisted high-throughput screening for organic room-temperature phosphorescent molecules

RTP

This project is created for high-throughput screening of promising room temperature phosphorescence (RTP). The training set is collected from the literature, which is expanded to serve as a better sample of chemical space. The dataset for screening is from PubChem, which is converted to 45 million valid SMILES and uploaded to (as the data is large). Note: users need to download and unzip the compressed file to the folder smiles_bit_pickle if screening is required.
